Setting up ghost

Hey! this post will guide you towards setting up your own blog using Ghost on github.

This blog is made using Ghost. I found ghost to be very useful for maitaining a blog.

How to begin installing and deploying it?

This blog will guide you to setting up ghost on github using gh-pages.

Requirements.

  • pip : python package manager (To install Buster).
  • Buster : Install it using pip install buster.
  • NodeJS : To install and run Ghost.
  • npm : NodeJS package manager.
  • knex-migrator : Setting up DB.

Steps:

  • Make a directory and install Buster using pip.
  • Download the Ghost and extract it inside the directory you just made.
  • I recommend using a LTS version of Ghost (currentlt v0.11.11).
  • Once extracted run npm install. And wait for dependencies to install.
  • Install knex-migrator and run knex-migrator init.
  • Run npm start to start the server and start browsing at localhost:2368.
  • You can access the admin page from localhost:2368/ghost.
  • And start adding your post.

Deploying to GitHub.

  • Make a repository on github and initialize it.
  • Run buster setup from your system.
  • And provide it your repo's git url.
  • Run buster generate to generate your static files.
  • Run Buster deploy to let it deploy to your repo.

You have successfully deployed to github pages. :+1:

Now You can access it from http://<your-github-username>.github.io/<your-repo-name>
